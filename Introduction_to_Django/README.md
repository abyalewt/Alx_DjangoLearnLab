# Alx_DjangoLearnLab

## Project Overview
This repository is dedicated to learning and implementing Django. It currently contains the **Introduction_to_Django** module, where I am practicing the basics of project structure, models, and CRUD operations.

## Current Project: LibraryProject
The `LibraryProject` is a Django-based application designed to manage a bookshelf. 

### Features:
- **Bookshelf App**: A custom app to manage book data.
- **Book Model**: Includes fields for `title`, `author`, and `publication_year`.
- **CRUD Operations**: Documentation of Create, Retrieve, Update, and Delete operations using the Django ORM.

## Directory Structure
- `bookshelf/`: Contains the model definitions for the book app.
- `LibraryProject/`: Contains the project settings and configuration.
- `*.md`: Evidence files for database operations (create, retrieve, update, delete).

## How to Run
1. Clone the repository.
2. Navigate to the `Introduction_to_Django` directory.
3. Run migrations: `python manage.py migrate`.
4. Start the shell: `python manage.py shell`.
